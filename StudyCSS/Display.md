<!-- Thuộc tính display: Thay đổi chế độ hiển thị của 1 thẻ html -->

- none -> Ẩn
- block -> Hiển thị dưới dạng block
- inline -> Hiển thị dưới dạng inline
- inline block -> Kết hợp inline và block

Lưu ý: Thuộc tính display có rất nhiều giá trị phức tạp

- flex, inline-flex
- grid, inline-grid
- table
- list-iem
- table-cell
